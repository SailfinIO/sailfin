This section collects informative design and rollout notes: wholly planned
features, syntax that parses before enforcement is complete, and partially
shipped surfaces whose remaining boundaries still matter. Each page labels
what works today and what remains future work. Normative language semantics
live in the [Language Specification](/docs/reference/spec/), while the
canonical implementation matrix lives in `docs/status.md`.

Proposals live under
[`docs/proposals/`](https://github.com/SailfinIO/sailfin/tree/main/docs/proposals)
in the repository; see the [Roadmap](/roadmap) for sequencing.
