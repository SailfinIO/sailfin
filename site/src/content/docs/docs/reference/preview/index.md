This section collects **planned** language features — syntax that may parse
today but is not yet enforced, and features that are on the roadmap but not
yet implemented. Treat these pages as informative. A feature only moves to
Part A of the [Language Specification](/docs/reference/spec/) once it is
fully implemented, tested, and documented in `docs/status.md`.

Proposals live under
[`docs/proposals/`](https://github.com/SailfinIO/sailfin/tree/main/docs/proposals)
in the repository; see the [Roadmap](/roadmap) for sequencing.
