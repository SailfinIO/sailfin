# Algorithms Examples

This folder contains classic algorithm implementations in Sailfin to illustrate control flow, recursion / iteration patterns, and type usage.

## Included

- **`quicksort.sfn`** – Demonstrates divide-and-conquer sorting, recursion, and list manipulation with functional style helpers.

## Planned Additions

Future examples may include: `merge-sort`, `binary-search`, `dijkstra`, `astar`, and streaming algorithms illustrating pipelines once the optimiser & effect checker mature.

> Contributions welcome—keep each example focused and annotated so it doubles as a learning artifact.
