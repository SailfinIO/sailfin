# Archive

Superseded, rejected, or stale historical proposal/work notes, preserved for the
record. **Nothing here is a live SFEP** — do not cite these as current design.

Examples of what lands here: superseded designs, completed one-off migration
audits, and old phase work-notes that fed a living tracker (e.g.
`docs/proposals/0006-build-architecture.md`) and are no longer current.

See [`../0001-sfep-process.md`](../0001-sfep-process.md) §9 for the lifecycle.
